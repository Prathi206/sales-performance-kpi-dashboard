# Sales Performance Dashboard

## Project Overview
This project analyzes Superstore sales data using SQL, Python, and Power BI to identify business insights, sales trends, profit analysis, and top-performing products.

## Tools Used
- SQL Server
- Python (Pandas, Matplotlib)
- Power BI

## Dataset
- Superstore Sales Dataset
- Cleaned dataset: `superstore_final.csv`

## Project Workflow
1. Data Cleaning using Python
2. SQL Queries for KPI analysis
3. Dashboard creation in Power BI
4. Business insights generation

## Dashboard Insights
- Total Sales
- Total Profit
- Profit Margin
- Regional Sales Analysis
- Top 10 Products
- Monthly Sales Trends

## Files Included
- `queries.sql` → SQL queries
- `superstore_analysis.ipynb` → Python analysis notebook
- `Sales_Performance_Dashboard.pbix` → Power BI dashboard
- `Sales_dashboard.png` → Dashboard screenshot
- `superstore_final.csv` → Final cleaned dataset

## Conclusion
This dashboard helps understand sales performance and business growth opportunities using data analytics techniques.
